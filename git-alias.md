# CREAR ALIAS

# Git config
```
git config --global alias.lodag 'log --oneline --decorate --all --graph'
git config --global alias.com 'commit'
git config --global alias.unstage 'reset HEAD --'
```

## Muestra todos los alias
```
git config --global --get-regexp alias
```
